# Modelo de Score de Crédito

## Objetivo
Desenvolver modelo preditivo de risco de crédito utilizando Regressão Logística e avaliar performance através de AUC, Gini e KS.

## Etapas
- Simulação de base de crédito
- Feature engineering
- Treinamento Logistic Regression
- Avaliação com AUC, Gini e KS
- Curva ROC
- Análise de coeficientes
- Simulação de cutoff de aprovação

## Métricas de Performance
- AUC
- Gini
- KS
- Taxa de aprovação
- Inadimplência dos aprovados

## Insight Executivo
O modelo permite ajustar políticas de crédito via cutoff, balanceando risco e crescimento da carteira.
